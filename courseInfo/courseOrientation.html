CS201: Course Orientation

Introduction

Welcome to CS201 – Computer Systems I. I am your instructor, Michael Trigoboff. I've been teaching here at PCC since 2002. Before teaching, I had over 30 years of experience as a software engineer, working on everything from Artificial Intelligence to disk drivers. You can find a more complete description of me on my PCC web site.

In CS201, we will go below the surface of your C source code, and learn about the object code produced by the compiler. When your program runs it is actually this object code that is running.

Then we will learn about some Unix operating system features that your object code can use. We will also descend into the computer chip itself, and discover some surprising and important things about how your object code is actually executed by the chip.

Software and Hardware Requirements

We will work extensively in Cygwin, which is a Windows-based Unix emulator. Cygwin has a source-level debugger called insight with a graphical user interface which will be especially helpful in understanding compiled code. You will need a Windows PC (or a Mac with Windows installed) to run Cygwin.

This term we will be exploring features of Unix that are not properly simulated by Cygwin. Since the demo code will not run properly in Cygwin, I will be running it in a real Linux system, syccuxfs01.pcc.edu. You will each be given an account on this system. For further instructions on how to access this account, see the What To Do Next section below.

Because of some limitations in the Unix software tools, I will occasionally use code written for Microsoft Visual C++ 2010 Express Edition. This is a free download from Microsoft. You can also use a computer at one of PCC’s Computer Resource Centers.

In order to connect with syccuxfs01.pcc.edu you will need an SSH client. One that works well on PC’s is PuTTY. Set up PuTTY to use SSH version 2 on the default SSH port (22). To transfer files back and forth between your PC and syccuxfs01.pcc.edu, I suggest that you use WinSCP.

Where To Find The Code

Many of the lessons in this course will refer to code examples. There will be links in each lesson to the code it uses.

Code that should run in Linux will be available on syccuxfs01.pcc.edu in directory ~mtrigobo/cs201. The subdirectory structure of this directory matches the structure of the lessons. Non-Linux code will be linked directly from the lessons so that you can download it to your machine.
