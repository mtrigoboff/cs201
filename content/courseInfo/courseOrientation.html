CS201: Course Orientation

Introduction

Welcome to CS201 – Computer Systems I. I am your instructor, Michael Trigoboff. I've been teaching here at PCC since 2002. Before teaching, I had over 30 years of experience as a software engineer, working on everything from Artificial Intelligence to disk drivers. You can find a more complete description of me on my PCC web site.

In CS201, we will go below the surface of your C source code, and learn about the object code produced by the compiler. When your program runs it is actually this object code that is running.

Then we will learn about some Unix operating system features that your object code can use. We will also descend into the computer chip itself, and discover some surprising and important things about how your object code is actually executed by the chip.

This course is often taught as parallel sections, one in the classroom and the other via distance learning. When that is the case, distance students who want to attend classroom sessions are always welcome.

Software and Hardware Requirements

The work for this course will be done in Linux. Linux has a source-level debugger called gdbtui with a graphical user interface which will be especially helpful in understanding compiled code.

PCC maintains a Linux system, syccuxfs01.pcc.edu. You will each be given an account on this system. For instructions on how to access this account, see the What To Do Next section below.

Because of some limitations in the Unix software tools, I will occasionally use code written for Microsoft Visual Studio 2013. This is a free download from Microsoft. You can also use a computer at one of PCC’s Computer Resource Centers.

In order to connect with syccuxfs01.pcc.edu you will need an SSH client. One that works well on Windows 7 is PuTTY. To transfer files back and forth between your PC and syccuxfs01.pcc.edu, I suggest that you use WinSCP.

Where To Find The Code

Many of the lessons in this course will refer to code examples. The code will be available on syccuxfs01.pcc.edu in directory ~michael.trigoboff/cs201/code. The subdirectory structure of this directory matches the structure of the lessons.
