CS201: GAS FSUB Bug

There is a bug in the GNU Assembler (GAS) that causes the wrong instruction to be produced when instructions like fsubp and are specified.

There is supposed to be a #define that you can set to get the correct instructions, but it seems not to work.

Unfortunately, it seems to have no effect.
