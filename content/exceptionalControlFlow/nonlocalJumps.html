CS201: Nonlocal Jumps

setjmp and longjmp

  • these functions are a part of the C Standard Library
  • setjmp()
    • saves the context at the point in execution where it’s called
    • returns 0 when called for the first time
    • returns the 2nd arg to longjmp() thereafter
    • what if we want to return 0 using longjmp?
  • longjmp()
    • “returns” to the instruction after the call to setjmp()
  • code examples
    • ljmp1.c: recursive implementation of factorial
    • ljmp2.c: recursive implementation of factorial that computes its result “on the way down”
    • ljmp3.c: uses longjmp(), calls setjmp() many times
    • ljmp4.c: uses longjmp(), only needs to call setjmp() once
      • ljmp4print.c: same as above, but prints out what it’s doing
    • ljmp5.cpp: uses C++’s more modern mechanism of throwing and catching exceptions
