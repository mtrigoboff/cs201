CS201: Binary and Hexadecimal Number Systems

Computers store numbers (and all other data as well) as collections of bits (which is an abbreviation for binary digits). A bit can assume one of two states: zero or one. When written in the usual way, as a decimal number, the number one hundred and seventy three is: 173. That same number would be written as a binary number (i.e. using bits) as: 10101101. That same number would be written as a hexadecimal number as: 0xAD. Hexadecimal numbers are also referred to as “hex” numbers.

Understanding these alternative number systems is required by the topics of this course. If you are not already familiar with these number systems (or if you want to refresh your knowledge), this discussion should be helpful.

Also note that the calculator that comes with Windows 7 can convert numbers between decimal, binary, and hexadecimal. The calculator has a very nice programming-oriented mode which you can access via the View menu.
