CS201: About This Module

What You Will Learn

Basic skills in the use of Linux and its gdb source‑level debugger.

An introduction to input/output for Unix command-line applications, including:

  • the use of printf() to write formatted output to the Unix window
  • the use of scanf() to read input typed by the user
  • how to access the command-line arguments that an application was invoked with
  • the use of atoi() and atof() to parse numerical command-line arguments
