ASCII stands for American Standard Code for Information Interchange. It is how character data is encoded in C strings, one byte per character. Standard ASCII character bytes have a leftmost bit of 0.

Standard ASCII was originally defined for machines like teletypes, that were based on a 7-bit byte.

There is also extended ASCII. These bytes have a leftmost bit of 1, and were used for things like drawing boxes on text-based terminal screens.
