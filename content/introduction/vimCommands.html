In the table below, [n] is an optional positive number. If n is not specified, 1 is assumed.

command explanation
Navigation
arrow keys up, down, left right
[n]w forward n word(s)
[n]b back n word(s)
g^g show current line number
gg go to first line
[n]G go to line number n
G go to last line
Insert Mode
i enter insert mode before cursor
a enter insert mode after cursor
I enter insert mode at start of cursor line
A enter insert mode at end of cursor line
O insert line above cursor, enter insert mode
o insert line below cursor, enter insert mode
escape exit insert mode
Editing
x or del delete character at cursor
rc replace character at cursor with c
cw delete word, enter insert mode
[n]dw delete n word(s)
[n]dd delete n line(s)
[n]>> indent n line(s)
[n]<< outdent n line(s)
J "join" cursor line with next line
u undo
"Cut" and "Paste"
[n]yw "yank" n words to "clipboard"
[n]yy "yank" n lines to "clipboard"
P insert "clipboard" above/before cursor
p insert "clipboard" below/after cursor
Search
/string search forward for string
/ search forward for previous string
?string search backward for string
? search backward for previous string
Substitute
:s/s1/s2/ change first occurrence of s1 to s2
:s/s1/s2/g change all occurrences on line of s1 to s2
:#1,#2s/s1/s2/g change all occurrences of s1 to s2 in lines #1 thru #2
:%s/s1/s2/g change all occurrences of s1 to s2 in file
:%s/s1/s2/gc change all occurrences of s1 to s2 in file, with confirmation
Subwindows (if no file specified, use current file)
:sp [file] horizontal split
:vs [file] vertical split
:e file edit a different file
Tab Windows
:tabe [file] new tab
gt next tab
gT previous tab
:mks setupFile save tab/subwindow setup
vim -S setupFile restore tab/subwindow setup
Saving and Quitting
ZZ save and quit current subwindow
:q quit subwindow
:qa quit all subwindows
:q! quit without saving
:qa! quit all subwindows without saving
:w save current subwindow
:wa save all subwindows
:wqa save and quit all subwindows
