CS201: GNU/Linux

In this course, we will be using the command-line Linux programming environment. Being able to operate in in this environment is an important skill, and PCC is providing you with the opportunity to acquire this skill in CS201. For students transferring to PSU, this will be especially important.

Linux should actually be called GNU/Linux, for reasons described here. GNU is an acronym for GNU’s Not Unix, which is an example of a recursive acronym. The pronunciation of GNU, and the history of recursive acronyms are described here by Richard Stallman, the founder of The GNU Project. GNU was the initial project in what eventually became the Open Source Software movement.
