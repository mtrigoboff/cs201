CS201: PuTTY Setup

For the Text User Interface (TUI) mode of the gdb debugger to work properly (i.e. subwindow frames defined by lines instead of garbage characters), you need to make some configuration settings in PuTTY.

In Window → Translation, you need to set the following:

Remote character set:ISO-8859-1: 1998 (Latin-1, West Europe)
Handling of line drawing characters:Use Unicode line drawing code points

If you download and expand this zip file, you will get a file named cs201syccuxas01.pcc.edu.reg. If you double-click this file it will add a session named syccuxas01.pcc.edu CS201 to PuTTY by adding a key to your Windows Registry.

This session has its configuration settings set up as described above.
