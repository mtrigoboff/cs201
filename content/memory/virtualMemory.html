CS201: Virtual Memory

In addition to reading about this topic in Bryant & O’Hallaron, look at these slides.

Virtual memory is a caching scheme that caches data from the disk in RAM. It is very similar to the caching described in the previous section. The mathematical term for this similarity is “isomorphic,” which means that there is a one-to-one correspondance between concepts in the two areas. A “cache” miss in the language of caching, for instance, is equivalent to a “page fault” in the language of virtual memory.

The difference between the language of virtual memory and the language of caching is an example of convergent evolution – such as when wolf-like animals evolved completely independently on two different continents.
