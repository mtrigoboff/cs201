CS201: The Memory Hierarchy

In addition to reading about this topic in Bryant & O’Hallaron, look at these slides.

There is more than one level of memory in a computer. Each level has its own characteristics. Ideally, we would have just one level of very large and fast memory. But that would be too expensive. So instead we have a memory hierarchy, a multi-level arrangement which allows us to use smaller amounts of the faster and more expensive memory, and larger amounts of the slower and cheaper memory.

The presence of this hierarchy in our computers causes large complications. But even considering the costs imposed by those complications, it’s still cheaper than a one-level system based on the fastest (but very expensive) memory.
